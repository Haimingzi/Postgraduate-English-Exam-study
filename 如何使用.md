# Cloze Test 使用说明（小白版）

这是一个**英语填空学习**的小应用：你输入一些单词，系统生成一篇短文，文中有几处空白；你从四个选项里选一个填进去，选完会立刻告诉你对错。

---

## 现在可以使用了吗？

可以。按下面**三步**做完就能用：

1. **安装依赖**：在项目目录 `cloze-test-app` 下打开终端，执行 `npm install`（等它跑完）。
2. **配置密钥**：在项目根目录建 `.env.local`，写一行 `DEEPSEEK_API_KEY=你的密钥`（密钥在 [Google AI Studio](https://aistudio.google.com/apikey) 免费申请）。
3. **启动**：执行 `npm run dev`，用浏览器打开 **http://localhost:3000**。

然后按下面「在应用里怎么用」操作即可。

---

## 一、先让应用跑起来

### 方式 A：在你自己电脑上运行（本地）

1. **安装 Node.js**  
   - 打开 [https://nodejs.org](https://nodejs.org)  
   - 下载并安装 **LTS 版本**（推荐 18 或 20）  
   - 安装完成后，打开「命令提示符」或「终端」，输入 `node -v`，能显示版本号就说明装好了  

2. **打开项目文件夹**  
   - 在终端里进入项目目录，例如：  
     ```bash
     cd 你的路径/cloze-test-app
     ```  

3. **安装依赖**  
   - 在项目目录下输入：  
     ```bash
     npm install
     ```  
   - 等它跑完（可能要一两分钟）  

4. **配置 API 密钥（必须，否则不能生成文章）**  
   - 在 [Google AI Studio](https://aistudio.google.com/apikey) 用谷歌账号登录，免费申请一个 **API Key**  
   - 在项目根目录新建一个文件，名字叫 `.env.local`（注意最前面有个点）  
   - 在 `.env.local` 里写一行（把下面的密钥换成你自己的）：  
     ```bash
     DEEPSEEK_API_KEY=你的API密钥
     ```  
   - 保存文件  

5. **启动应用**  
   - 在项目目录下输入：  
     ```bash
     npm run dev
     ```  
   - 看到类似 `Local: http://localhost:3000` 后，用浏览器打开 **http://localhost:3000**  

---

### 方式 B：用 Vercel 在线使用（推荐，不用装 Node）

1. 把项目代码推到 **GitHub**（如果还不会，可以搜「GitHub 上传代码」按教程做）  
2. 打开 [vercel.com](https://vercel.com)，用 GitHub 登录  
3. 点 **Add New… → Project**，选择你这个项目的仓库，然后 **Deploy**  
4. 部署好后，在 Vercel 里进入项目的 **Settings → Environment Variables**  
5. 新增一个变量：**Name** 填 `DEEPSEEK_API_KEY`，**Value** 填你在 Google AI Studio 申请的 API Key，保存  
6. 到 **Deployments** 里对最新一次部署点 **⋯ → Redeploy**，等部署完成  
7. 打开 Vercel 给你的网址（例如 `https://xxx.vercel.app`），就可以在浏览器里用这个应用了

## 二、在应用里怎么用

### 1. 输入单词

- 在页面顶部的**大文本框**里输入你想练习的英语单词  
- 可以：  
  - **一行一个**，例如：  
    ```text
    vocabulary
    reading
    improve
    ```  
  - 或用**逗号、空格**分隔，例如：`word, test, learn`  

### 2. 生成文章

- 输入完单词后，点击下方的 **「生成」** 按钮  
- 会先出现一段**灰色骨架动画**（表示正在生成）  
- 几秒到十几秒后，下面会出现一篇**英文短文**，里面有些地方是 **\[ 1 \]**、**\[ 2 \]** 这样的空白  

### 3. 做题（选答案）

- 文中**你输入的单词**会被**黄色高亮**，方便你看到「你的词用在这里」  
- 点击任意一个 **\[ 1 \]**、**\[ 2 \]** 等空白  
- 屏幕**底部会弹出一个选择框**，里面有 **A、B、C、D 四个选项**  
- 点一个选项，就相当于「填了这个空」  
- 选完后：  
  - **选对了**：这个词会变成**绿色**  
  - **选错了**：这个词会变成**红色**  
- 选完还会在文章下面出现一张**说明卡**，告诉你「第几空」、你的答案、以及正确答案  

### 4. 继续做下一空

- 再点其他 **\[ 3 \]**、**\[ 4 \]** 等空白，同样从底部弹窗里选 A/B/C/D即可  
- 已经选过的空会**直接显示你选的词**（绿色或红色），一般不能重复再选  

### 5. 换一批题

- 想重新生成一篇新文章时：可以**改一下单词列表**（或清空再输入新的），再点一次 **「生成」**  

---

## 三、常见问题

- **点「生成」没反应或报错？**  
  - 检查是否已经配置了 `.env.local` 里的 `DEEPSEEK_API_KEY`（本地）或 Vercel 里的环境变量（线上）  
  - 密钥要有效、没有多余空格  

- **文章里的空白点不了？**  
  - 那个空可能已经选过答案了（会显示成绿色或红色的词），已选过的空不能再点  

- **想用手机打开？**  
  - 如果用 Vercel 部署了，用手机浏览器打开 Vercel 给你的网址即可；选项会固定在屏幕底部，方便手指点  

---

总结：**输入单词 → 点「生成」→ 等文章出现 → 点空白 → 在底部选 A/B/C/D → 看对错和说明**。多练几遍就熟了。
